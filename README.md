# 🚀 React + NestJS Authentication App

A simple full-stack authentication project built with:

- Frontend: React + Tailwind CSS + Axios  
- Backend: NestJS (No database, hardcoded users)

---------------------------------------------------------------------

## 📁 Project Structure

root/
├── frontend/ # React App
└── backend/ # NestJS API

---------------------------------------------------------------------

## ⚙️ Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios

### Backend
- NestJS

---------------------------------------------------------------------

## 🖥️ Frontend Setup

### 📌 Features
- Login Screen
- Dashboard Screen
- Authentication handling
- API integration using Axios

### ▶️ Run Frontend

```bash
cd frontend
npm install
npm run dev

---------------------------------------------------------------------
